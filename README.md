# Task Tracker

A simple desktop task management app built with Python and Tkinter.

## Features

- Add tasks using a textbox
- Display tasks in a listbox
- Simple and clean GUI
- Beginner-friendly project structure

## Technologies Used

- Python
- Tkinter

## How It Works

1. Type a task into the input box
2. Click the **Add Task** button
3. The task appears in the task list
